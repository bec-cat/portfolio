Role: Visual Design, Hardware Research, Prototyping

Timeline: 12 weeks (Feb-May 2021)

Tools: Arduino IDE

Overview

The final prototype is a walking stick with an ultrasonic sensor to assist object detection, and an accelerometer to function as a pedometer, using Arduino uno together with an app made with the help of open source code.

This project was created as part of prototyping physical interaction course in a group of 10 people; 10 researchers, 3 programmers, 4 product designs, 3 user testings. The design brief for this project involves physical game such as activity and movement.

Aims

The aim was to create a prototype device to assist vision impaired individuals with navigating to new places and encouraging physcial exercise through the use of a built in pedometer.

  • Ascertain feasibility of prototype design for everyday use.
  • Test usability of proposed design and solution.
  • Determine if product assisted in improving the person's object detection.
  • Determine if the user finds the game/activity enjoyable.
